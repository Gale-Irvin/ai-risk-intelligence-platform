# Executive Risk Intelligence Dashboard Layout

## Top Summary Metrics
- Total Active Risks
- High / Medium / Low Risks
- Top Event Type
- Most Impacted Business Function

## Main Panels
### 1. Risk Summary Table
Source: Dashboard_Risks.csv

Purpose:
Show each risk by event type, asset, business function, confidence, likelihood, impact, and risk score.

### 2. Risk Trend Chart
Source: Dashboard_RiskTrends.csv

Purpose:
Show weekly volume of risk signals by event type and asset.

### 3. Risk Evidence Table
Source: Dashboard_RiskEvidence.csv

Purpose:
Provide traceability to the articles and reports driving each risk signal.

### 4. Event Type Breakdown
Source: Dashboard_Risks.csv

Purpose:
Summarize the number of risks by event type.

## Executive Use
This dashboard supports operational resilience leadership by translating external incident signals into structured risk visibility for executive decision-making.
